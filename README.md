# Movies Catalog (IMDb-like) — Frontend

A small frontend-only movies/series/cartoons catalog app built with Create React App, Ant Design and TMDb API.

## Features
- 6 pages:
  - Home (Trending mixed titles + Search)
  - Movies (Discover + filters)
  - Series (Discover + filters)
  - Cartoons (Animation movies)
  - Title Details (poster + text info + cast)
  - My Catalog (favorites)
- Cards: poster, title, genres, year, rating
- Favorites + personal rating (1–10) stored in LocalStorage (no backend)
- Filters: genre, year
- Sorting: alphabet, year, rating
- Pagination via "Load more"

## Tech Stack
- React (Create React App)
- Ant Design
- Axios
- React Router
- TMDb API
